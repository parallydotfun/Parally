Dynamic Background Effect

Move your cursor around the screen to see the gradient effect follow your movement in real-time. The subtle grid pattern adds depth while maintaining perfect readability.

Smooth Tracking

The gradient follows your cursor with smooth interpolation using linear interpolation (lerp) for a natural, fluid feel without jarring movements.

Grid Pattern

Subtle 50px grid overlay at 3% opacity adds visual depth and structure without overwhelming the content or reducing readability.

Performance Optimized

Built with requestAnimationFrame for smooth 60fps rendering. No performance impact even on lower-end devices with proper GPU acceleration.

Technical Specifications

  • Base Color: Pure black (#000000) for maximum contrast
  • Grid Pattern: 50px squares with 3% white opacity
  • Gradient: 600px radial at 15% opacity using screen blend mode
  • Animation: Linear interpolation with 0.15 smoothing factor
  • Performance: requestAnimationFrame ensures optimal 60fps
  • Compatibility: Works in all modern browsers (Chrome, Firefox, Safari, Edge)
  • Accessibility: Maintains WCAG contrast ratios for text readability
